# Development

Install local packages

```bash
npm install
```

Start dev server on localhost:3000

```bash
npm run dev
```

Start prod server on localhost:3000

```bash
npm start
```

Lint and test

```bash
npm run lint:fix && npm test
```

# Build

```bash
npm run build
```
